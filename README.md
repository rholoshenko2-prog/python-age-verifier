# Проверка возраста

Простая консольная программа на Python, которая проверяет возраст пользователя и выдаёт доступ с учётом правильного склонения слов «год/года/лет».  
Написана на День 4 обучения Python (глава про условные операторы if/else/elif).

## Что умеет программа

- Запрашивает имя и возраст
- Проверяет, что возраст введён цифрами
- Если возраст ≥ 18 → разрешает доступ и приветствует
- Если возраст < 18 → считает, сколько лет осталось и правильно склоняет «год/года/лет»
- Выводит красивый лог с результатом проверки

## Как запустить

1. Установи Python (если ещё не установлен)
2. Сохрани код в файл, например age_check.py
3. Открой терминал / командную строку в папке с файлом
4. Запусти:
python age_check.py
